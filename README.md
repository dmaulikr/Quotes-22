Quotes for iOS

Build dependencies via [Carthage](https://github.com/Carthage/Carthage) first:

``` shell
$ carthage bootstrap
```

## Backend

Build dependencies via [Bundler](http://bundler.io) first:

``` shell
$ cd Backend
$ bundle install
```

To start the server use [Shotgun](https://github.com/rtomayko/shotgun):

``` shell
$ cd Backend
$ bundle exec shotgun
```
